# Warehouse Inventory Alert System

## Project Summary
This project is a console-based Warehouse Inventory Alert System developed using C++. It helps manage products in a warehouse by allowing add, delete, update, search, display, and low-stock alert operations.

## Data Structures Used
- Singly Linked List: Used to store product records dynamically.
- Stack: Used to implement undo functionality using LIFO principle.

## Key Features
- Add, delete, update, and search products
- Low stock alert system
- Undo last operation
- Dynamic memory management
- Menu-driven interface

## How to Run
1. Compile the program using a C++ compiler.
2. Run the executable.
3. Use the menu options to manage inventory.
